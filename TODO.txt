
Идентификация объекта производится в шейдере при помоши записи сразу в два
буфера. В служебный буфер записывать не цвет, а значение

// Кодирование целого ID в части RGB

int r, g, b = gl_PrimitiveID;

r  = b % 255;
b /= 255;
g  = b % 255;
b /= 255;

// В шейдере запись производится при помощи вызовов

gl_FragData[0] = vec4(r, g, b, 0);   // служебный буфер
gl_FragData[1] = color; // основной текстурный буфер

// Декодирование в целое:
int ID = r + (g + b * 255) * 255;
