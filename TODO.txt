
Отображение боковых стенок:
---------------------------
  Хранить в ригах не плоскоские снипы, а объемы по 8 вершин.
  Отдельно хранить стыки ("splice") для всех 6 сторон.
  - Сторона невидимая в том случае, если сплайсы у соседних ригов равны
  - Во всех остальных случаях сторона отображается.

  В spliceXp, spliceYp, spliceZp делать pushBack
  B spliceXn, spliceYn, spliceZn делать pushFront

алгоритмы
-------------------------------------------------------------------------------
// Кодирование целого ID в части RGB

int r, g, b = gl_PrimitiveID;

r  = b % 255;
b /= 255;
g  = b % 255;
b /= 255;

// В шейдере запись производится при помощи вызовов

gl_FragData[0] = vec4(r, g, b, 0);   // служебный буфер
gl_FragData[1] = color; // основной текстурный буфер

// Декодирование в целое:
int ID = r + (g + b * 255) * 255;
