
Настроить работу с пространством, загружаемым из базы данных.

  1. При завершении работы в базе данных сохраняются координаты текущего
    положения камеры.

  2. При загрузке сцены следует передавать это координаты в метод инициализации
    карты пространства (файл "cpace.cpp", строка 50:
```
    RigsDb0.init(g0); // загрузка данных уровня LOD-0 )
```

чтобы он загружал данные из базы выбранной локации, и только при их отсутствии
генерировал карту пространства по шаблону.

Идентификация объекта производится в шейдере при помоши записи сразу в два
буфера. В служебный буфер записывать не цвет, а значение

// Кодирование целого ID в части RGB

int r, g, b = gl_PrimitiveID;

r  = b % 255;
b /= 255;
g  = b % 255;
b /= 255;

// В шейдере запись производится при помощи вызовов

gl_FragData[0] = vec4(r, g, b, 0);   // служебный буфер
gl_FragData[1] = color; // основной текстурный буфер

// Декодирование в целое:
int ID = r + (g + b * 255) * 255;
